GPS Tracking System — Modern UI Final

สิ่งที่แก้:
- ใช้ไอคอนรถสีขาว + หมุด GPS ตามรูปที่เลือก
- เอาการ์ด “อัปเดตล่าสุด” ออกจาก Dashboard
- เหลือ Dashboard 5 รายการ: การเชื่อมต่อ, โหมดรถ, ความเร็ว, ดาวเทียม, ระยะทาง
- รวมการแสดง/ซ่อนเส้นทางไว้ในปุ่มเดียว และข้อความปุ่มเปลี่ยนตามสถานะ
- เก็บเวลาอัปเดตล่าสุดไว้ในรายละเอียดและ Popup
- ไม่แก้ Logic GPS, Firebase, ความเร็ว, สถานะรถ หรือรอบการอ่านข้อมูล
