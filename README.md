## วิธีอัปเดต GitHub Pages

1. แตก ZIP
2. อัปโหลดไฟล์ทั้งหมดไว้ใน root ของ repository `GPS-Tracker`
3. เข้า Settings > Pages
4. Source: Deploy from a branch
5. Branch: main / root
6. เปิดลิงก์ GitHub Pages

## การใช้งานบนมือถือ

เปิดลิงก์ GitHub Pages ด้วย Chrome/Safari แล้วเลือก Add to Home Screen หรือ Install App
